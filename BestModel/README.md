# Best Model (Submission)

This folder contains the best-performing PCL classifier for submission.

## Contents

- **`model.ipynb`** — Training and inference notebook (run from this directory or repo root)
- **`roberta-base/`** — Best checkpoint: RoBERTa-base, weight_decay=0.01, dropout=0.1
  - `config.json`, `tokenizer.json`, `tokenizer_config.json`
  - Model weights (`pytorch_model.bin` or `model.safetensors`) are generated by training. If missing, run `4_model/model.ipynb` with `TRAIN_MODE=True`, then copy the best checkpoint from `BestModel/training/` into `roberta-base/`

## Training Code

The full training pipeline is in **`model.ipynb`** (this folder) and **`4_model/model.ipynb`**. Run from the repository root with:

1. `TRAIN_MODE = True` — trains from scratch and saves best checkpoints to `BestModel/training/`
2. `TRAIN_MODE = False` — loads existing checkpoints from `BestModel/roberta-base` and generates `dev.txt` / `test.txt`
